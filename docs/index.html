Spring Boot AI Agent Toolkit

A production-ready Spring Boot Starter for integrating AI agents, RAG pipelines, and MCP tools into Java applications.

⭐ GitHub 📦 Releases

✨ Features

🤖

Multi-Provider AI

Unified API for OpenAI (GPT-4, GPT-4o), Anthropic (Claude 3/4), and Ollama (local LLMs).

🔧

Tool Calling

Register Java methods as AI-callable tools with automatic JSON Schema parameter definitions.

📚

RAG Pipeline

Built-in document chunking, embedding, vector storage, and cosine similarity search.

🔌

MCP Integration

Connect to any MCP-compatible server for dynamic tool discovery and execution.

⚡

Streaming

Real-time response streaming for both chat and tool-calling scenarios.

🎯

Auto-Configuration

Spring Boot auto-configuration with sensible defaults. Zero XML, minimal YAML.

🚀 Quick Start

Add dependency

// build.gradle.kts
implementation("com.aiagentkit:spring-boot-ai-agent-toolkit:1.0.0")

Configure

# application.yml
aiagent:
  provider: ollama          # openai, anthropic, or ollama
  model: llama3.2:latest
  ollama-base-url: http://localhost:11434

Use it

@RestController
public class ChatController {

    @Autowired
    private AiAgent aiAgent;

    @PostMapping("/chat")
    public String chat(@RequestBody String message) {
        return aiAgent.chat(message);
    }
}

┌─────────────────────────────────────────────┐
│           Your Spring Boot App               │
├─────────────────────────────────────────────┤
│  AiAgent (Orchestrator)                      │
│  ├── AiProvider (OpenAI / Anthropic / Ollama) │
│  ├── ToolExecutor[] (Your Java Tools)        │
